**personal-tech-portal** 是基于 Astro Content Collections 构建的个人技术门户，定位于展示 AI / Cloud / DevOps 工程师的工作成果与技术思考。

- 三类内容一体管理：项目项目卡片、上线动态、技术博客
- 零 CMS 纯 Git 写流程，Markdown 即内容
- 一键部署到 Cloudflare Workers，无需服务器
- TypeScript 类型安全 Content Schema，IDE 感知力满分
