**copilot-agent-lite** 是一个基于 Python + FastAPI 的超轻量 Copilot subscription agent runtime。

- YAML 驱动 agent 配置，开箱即用
- APScheduler cron 定时任务（`schedules.yaml`）
- MCP server 配置透传，无缝扩爪牙
- Web UI 内置 agent 列表、任务状态、执行日志
- 轻量部署：Docker 或 pip install
