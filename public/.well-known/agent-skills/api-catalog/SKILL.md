# API Catalog

The site publishes an RFC 9727-style API catalog at:

`https://bbfans.github.io/.well-known/api-catalog`

The catalog points to:

- OpenAPI description: `https://bbfans.github.io/openapi.json`
- API documentation: `https://bbfans.github.io/docs/api/`
- Public status endpoint: `https://bbfans.github.io/api/status.json`
